# /analyze-cohorts -- Cohort Analysis

把用户按时间或行为分 cohort，分析留存、转化与行为差异。

## 调用方式

```text
/analyze-cohorts 注册用户留存
/analyze-cohorts 首次支付后的复购情况
```

## 工作流

### 步骤 1: Accept Data or Define Analysis
### 步骤 2: Define Cohorts
### 步骤 3: Analyze
### 步骤 4: Generate 报告

```markdown
## Cohort Analysis: [描述]
### Retention Table
### Key Findings
### Cohort Comparison
### Retention Benchmarks
### Recommendations
### Follow-Up Queries
```

### Step 5: Offer 下一步

- 继续写 SQL
- 补 dashboard
- 结合 experiment 继续分析

## Notes

- cohort 定义要与业务问题匹配
