# pm-go-to-market 中文整理版

GTM 插件，覆盖 ICP、beachhead segment、growth loops、GTM motions 与完整 launch strategy。

## Skills（6）

- `beachhead-segment`：识别产品最初的滩头市场
- `competitive-battlecard`：生成面向销售的竞品 battlecard
- `growth-loops`：识别可持续增长飞轮
- `gtm-motions`：选择合适的 GTM motion 与渠道组合
- `gtm-strategy`：制定完整的 go-to-market strategy
- `ideal-customer-profile`：识别并定义 ICP

## Commands（3）

- `/pm-go-to-market:battlecard`
- `/pm-go-to-market:growth-strategy`
- `/pm-go-to-market:plan-launch`

## 适用场景

- 产品准备从 0 到 1 进入市场
- 需要明确首个细分市场与 ICP
- 需要把增长机制与发售计划串起来
