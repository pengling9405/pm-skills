# Growth Loops

## 概览

增长循环强调“一个用户行为如何带来下一个用户或下一次使用”。

## When to Use

- 设计可持续增长机制
- 判断产品是否具备自增强增长潜力

## The 5 Growth Loop Types

### 1. Viral Loop
### 2. Usage Loop
### 3. Collaboration Loop
### 4. 用户-Generated Loop
### 5. Referral Loop

## Input Format

- 产品
- 当前增长目标
- 用户行为模式

## Output

- 推荐循环
- 循环结构
- 风险与前提

## Framework

优先找“用户自然行为”能驱动的循环，而不是硬造邀请机制。

## Tips

- 没有产品价值基础的 loop 很难持续
