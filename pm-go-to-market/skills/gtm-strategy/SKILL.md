# GTM 策略

## 概览

这个 skill 用于把 ICP、定位、渠道和发布策略整合成一份可执行 GTM 方案。

## When to Use

- 产品或功能上线前
- 想系统规划渠道与信息
- 需要 launch playbook

## How It Works

### 步骤 1: Gather Research Data
### 步骤 2: Define Marketing Channels
### 步骤 3: Develop Messaging
### 步骤 4: Define Success Metrics
### Step 5: Create Launch 计划

## Input Format

- 产品 / 功能
- ICP
- 发布目标
- 约束条件

## Output

- GTM 计划
- 渠道策略
- 关键 messaging
- 时间线与指标

## Framework

先锁定 beachhead 与 ICP，再设计渠道与发布节奏。

## Tips

- 信息与渠道必须围绕同一受众展开
