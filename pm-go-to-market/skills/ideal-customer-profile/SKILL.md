# Ideal Customer Profile

## 概览

这个 skill 用于识别最值得服务、最可能付费、最容易成功的客户类型。

## When to Use

- 做 ICP 定义
- 准备 GTM
- 细化销售与营销目标人群

## ICP Framework Components

### Demographics
### Behaviors
### Jobs to Be Done (JTBD)
### Needs and Pain Points

## How It Works

### 步骤 1: Gather Customer Data
### 步骤 2: Segment by Value
### 步骤 3: Profile Demographics
### 步骤 4: Identify Behaviors
### Step 5: Define JTBD
### Step 6: Document Pain Points and Needs

## Input Format

- 已有客户或候选市场数据
- 研究材料
- 销售 / 支持反馈

## Output

- ICP 描述
- 典型 JTBD
- 主要痛点
- 适合的 messaging 方向

## Framework

ICP 不是“任何可能买的人”，而是“最值得优先服务的人”。

## Tips

- 把“高价值客户”和“最容易成交客户”区分开看
