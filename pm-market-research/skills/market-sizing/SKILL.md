# Estimate Market Size (TAM, SAM, SOM)

## Purpose

帮助用户对市场机会建立量级判断，而不是拍脑袋估算。

## Instructions

### Input

- 产品或市场
- 目标用户段
- 价格 / 商业模式（如已知）

### Analysis Steps (Think Step by Step)

1. 定义市场边界
2. 估算 TAM
3. 缩到可服务市场 SAM
4. 再缩到现实可赢得份额 SOM

### Output Structure

- 假设
- 计算方法
- TAM / SAM / SOM
- 风险与不确定性

## Best Practices

- 明确假设来源
- 用区间而不是伪精确数字
