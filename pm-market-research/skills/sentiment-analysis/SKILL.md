# Sentiment Analysis

## Purpose

把用户反馈中的情绪模式结构化，帮助识别问题强度与情绪集中点。

## Instructions

### Input

- 评论、问卷、工单、反馈文本

### Analysis Steps (Think Step by Step)

1. 判断整体情绪
2. 识别正负主题
3. 找出高强度反馈
4. 结合用户段做差异分析

### Output Structure

- 整体情绪
- 高频正向 / 负向主题
- 代表性 quote
- 可执行洞察

## Best Practices

- 情绪只是线索，不是最终结论
- 要结合主题与用户段一起看
