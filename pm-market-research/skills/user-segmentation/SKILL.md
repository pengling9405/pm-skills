# User Segmentation

## Purpose

这个 skill 用于把“用户整体”拆成更有决策价值的 segment。

## Instructions

### Input

- 研究材料
- 反馈数据
- 使用行为

### Analysis Steps (Think Step by Step)

1. 找出明显分层模式
2. 依据任务、行为、需求做分群
3. 比较不同 segment 的价值与差异

### Output Structure

- 主要 segments
- 每个 segment 的定义与特征
- 推荐优先服务的 segment

## Best Practices

- 分群必须服务决策，不要只做静态分类
