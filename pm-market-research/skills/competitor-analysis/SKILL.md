# Competitor Analysis

## Purpose

帮助用户理解市场竞争格局，以及自身可以建立怎样的差异化位置。

## 使用说明

### Input

- 产品或市场名称
- 竞品名单（可选）
- 希望支撑的决策

### Analysis Steps (Think Step by Step)

1. 识别直接与间接竞品
2. 分析价值主张、用户、定价与 GTM
3. 比较关键优劣势
4. 找出差异化机会与威胁

### Output Structure

- 市场概览
- 竞争格局
- 功能 / 定位对比
- 差异化建议

## Best Practices

- 看用户为什么真的切换
- 看替代方案，而不只看同类产品
