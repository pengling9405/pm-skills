# /proofread -- Grammar & Flow Check

对文本做轻量审校，识别语法、逻辑和表达流畅性问题。

## Invocation

```text
/proofread [粘贴一段文本]
```

## Workflow

### Step 1: Accept Text
### Step 2: Analyze
### Step 3: Report Issues

```markdown
## Proofread Report
### Issues
#### 1. [Category: Grammar/Logic/Flow]
### Summary
```

### Step 4: Offer

- 继续重写
- 保留原意做 polish

## Notes

- 优先指出问题，不要默认整段重写
