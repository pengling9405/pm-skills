# /审查-resume -- PM Resume 审查

从 PM 岗位视角审阅简历，输出结构化改进建议。

## 调用方式

```text
/review-resume [上传简历]
```

## 工作流

### 步骤 1: Accept the Resume
### 步骤 2: Evaluate Against 10 Best Practices
### 步骤 3: Generate 审查

```markdown
## Resume 审查
### Scorecard
### Top 3 Improvements
### Section-by-Section Feedback
### Missing Elements
### Keywords to Add
```

### 步骤 4: Offer 下一步

- 针对 JD 定制
- 重写 summary
- 强化项目 bullet

## Notes

- 优先提升结果表达、结构与关键词匹配
