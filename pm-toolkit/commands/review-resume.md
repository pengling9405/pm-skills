# /review-resume -- PM Resume Review

从 PM 岗位视角审阅简历，输出结构化改进建议。

## Invocation

```text
/review-resume [上传简历]
```

## Workflow

### Step 1: Accept the Resume
### Step 2: Evaluate Against 10 Best Practices
### Step 3: Generate Review

```markdown
## Resume Review
### Scorecard
### Top 3 Improvements
### Section-by-Section Feedback
### Missing Elements
### Keywords to Add
```

### Step 4: Offer Next Steps

- 针对 JD 定制
- 重写 summary
- 强化项目 bullet

## Notes

- 优先提升结果表达、结构与关键词匹配
