# pm-marketing-growth 中文整理版

产品营销与增长插件，覆盖营销点子、North Star、定位、命名与价值主张表达。

## Skills（5）

- `marketing-ideas`：生成低成本、可执行的营销想法
- `north-star-metric`：定义 North Star Metric 与输入指标
- `positioning-ideas`：围绕竞品做差异化定位头脑风暴
- `product-name`：生成产品命名方案
- `value-prop-statements`：生成适用于营销、销售和 onboarding 的价值主张表达

## Commands（2）

- `/pm-marketing-growth:market-product`
- `/pm-marketing-growth:north-star`

## 适用场景

- 需要一句清晰定位
- 需要建立产品叙事与增长指标体系
- 需要从 PM 视角和营销视角同时看产品表达
