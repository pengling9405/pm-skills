## Product Metrics Dashboard

### 背景

这个 skill 用于设计一个能支撑产品决策的 dashboard，而不是单纯堆指标。

### Domain 背景

请区分：
- 输入指标
- 结果指标
- 健康指标
- 反向指标

### 使用说明

1. 明确业务目标
2. 选出核心指标与 supporting metrics
3. 设计 dashboard 结构
4. 为关键指标设置阈值与 review cadence
