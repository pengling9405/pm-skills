## Summarize Customer Interview

### 背景

这个 skill 用于把原始访谈记录转成可用于产品决策的结构化摘要。

### 使用说明

1. 提炼关键洞察
2. 识别用户任务、现有流程与痛点
3. 标记满意度与不满信号
4. 列出后续行动建议

### Output Template

```markdown
## Interview 摘要
### Key Insights
### Jobs to Be Done
### Current 工作流
### Pain Points
### Satisfaction Signals
### Notable Quotes
### Assumptions Validated / Invalidated
### Action Items
```
